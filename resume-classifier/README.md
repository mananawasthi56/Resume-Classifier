
# Intelligent Resume Classifier & Skill Extractor

**What it is:** A demo end-to-end project that cleans resume text, extracts skills, and classifies resumes into roles.

## Quick start (Streamlit)
```
cd resume-classifier/streamlit_app
pip install -r requirements.txt
streamlit run app.py
```

Power BI: preview image included in powerbi/ and instructions to recreate and publish.
